B. TRON Support Network 宣言



TRON Support Network
--------------------

                                        内藤 隆一
                                        (night@tron-net.gr.jp)	


1. 目的


1.1 現在の問題

  1984 年に TRON プロジェクトが発足してから、10 年以上経過している。
  その間にメーカサイドだけではなく、TRON に関心をもつ人たちによるプロ
ジェクトがいくつか進められてきた(ex. B-Free project)。			
  これらのプロジェクトは、互いに異なった方法で情報を公開しており、すべ
てのプロジェクトの情報を網羅するのは困難な状態にある。	
  たとえば、ある TRON のファンが特定のプロジェクトに参加している場合で
も、他の TRON 関係のプロジェクトがあることを知ることは人づてに聞くしか
なかった。また、すべてのプロジェクトの状況を知るには、各プロジェクト毎
に異なった方法でアクセスする必要があった。	

  このような状況では、TRON プロジェクトに関心のある人にとって、興味が
あるプロジェクトがあることを知らないままでいるケースが少なくないと思わ
れる。また、各プロジェクト間での協力体制の構築も難しいという問題がある。

  TRON Support Network は、以上の問題を解決するために、設立したもので
ある。


1.2 TRON Support Network の目的

  TRON support network とは、TRON プロジェクトに関心をもつ人たちに対し
て情報交換および情報の公開のために共通に使えるインフラを提供するネット
ワークである。								

  本ネットワークは、以下の目的をもつ。

  (1) TRON に関心をもつ人たちが行っている活動のためのインフラの提供
  (2) TRON に関心をもつ人たちの間の情報交換
  (3) TRON に関心をもつ人たちが行っている活動についての情報の公開

  これらの目的を達成するために、TRON support network では、インターネッ
トに常時接続したサイトを構築し、そのサイトを情報交換のための拠点とする。

  独自に TRON に関係したプロジェクトを立ち上げる意欲があるが、情報を発
信する術を持っていない人たちもいる。そういう人たちに対して、TRON
support network では情報発信のためのインフラを提供する。



2 TRON Support Network が提供するサービス

  TRON support network が立ち上げるインターネットのサイトでは、TRON に
関心をもつ人たちに対して次のサービスを提供するものとする。

  (1) 各プロジェクトから WWW による広い範囲への情報の公開手段の提供
  (2) プロジェクト内・プロジェクト間のMLによる情報交換
  (3) 各プロジェクト関係者へのメイルアカウントの提供


2.1 WWW による広い範囲への情報の公開

  インターネット = WWW の認識が広がっているように、WWW は多くの人々が
注目している新しいメディアになりつつある。
  すでに TRON に関心をもつ人たちの間でも、WWW による情報の提供が多く行
われている。だが、WWW による情報は、そのサイトへの URL が分からない限
りアクセスできないという欠点がある。そのため TRON ファンによる WWW の
立ち上げが行われていても、他のサイトからのリンクがないと、見る人が少な
い(逆に、ある程度知られるようになると、アクセス数は格段に増える)。
  TRON support network では、こうした埋もれたサイトへのリンクを張るこ
とによって、TRON に関心をもっている人たちの活動を多くの人の目につくよ
うにする。

  現在の TRON に関心をもつ人たちの活動は、個々人による情報の公開が行わ
れつつある状況にある。しかし、それらの活動は分散しているため、互いに
情報交換を行っている例は少ない。また、WWW による情報の公開を行うために
はネットワークについての知識がある程度必要になる。そのための手間を取れ
ない(とりたくない)人たちのために対しても WWW による掲示板などの場を提
供し、インターネット上へ情報を公開できるようにする。


2.2 ML による情報交換の支援

  TRON support network では、TRON のファンが行うプロジェクトについて、
必要ならば ML の立ち上げ支援および運営のための資源を提供する。

  TRON に関心もった人たちが独自の活動を行う上で大変なのは、メンバーと
の情報のやりとりのためのインフラ構築である。インターネットメイルを使っ
た ML (メイリングリスト) による情報のやりとりは、メンバーを特定の場に
縛られることがない。

  特に最近の BBS のほとんどがインターネットメイルが reachable であるこ
とを考えると、情報交換に使うには優れた方法だと思える。

  しかし、BBS に比べると ML を立ち上げるには、それなりの手間が必要にな
り、経験がない人にとっては大変な作業となる。TRON support network が ML
の立ち上げを肩代りすることによって、各プロジェクト内およびプロジェクト
間の情報交換をスムーズに行えるようになる。


2.3 各プロジェクト関係者へのメイルアドレスの提供

  TRON support network では TRON に関して独自に活動を進めている人に対
しては、メイルアドレスを提供するというサービス(メイルの中継サービス)を
行う。

  たとえば、linux.or.jp というサイトでは、Linux に対して貢献した人に対
して linux.or.jp のメイルアカウントを与えることが行われている。それら
の人たちに対してメイルを出す時には、何々@linux.or.jp というメイルアド
レスで届き、非常に分かりやすくなっている(これには別に linux.or.jp とい
う物理的実体によって Linux の活動が集中的に行われているように見える効
果がある)。  

  また、メイルアドレスはそれぞれの事情によって変更する場合がある。
そうした場合でも、TRON support network を中継していれば、TRON support
network の設定を変更するだけでよい。メイルを出す人は最終的に到着するメ
イルアドレスが変更になったかを気にすることなく、TRON support network
へのメイルアドレスにメイルを出すことができる。

  Linux ユーザの活動に限らず、FreeBSD などでもメイルアカウントの提供は
おこなわれている。FreeBSD の場合には、たとえば、日本人の場合には、何々
@jp.freebsd.or.jp というメイルアドレスを提供している。

  メイルアカウントの提供は、マシンへのログイン権限を与えるというわけで
はなく、メイルの転送サービスを行っているにすぎず、提供する手間はほとん
どない。


3. まとめ

  TRON support network は、TRON に関心をもつすべての人たちに対して開か
れたコミュニケーションのための場を構築する手段を与える。

  特に、インターネットというメディアに集中して情報交換および情報公開の
ためのインフラを提供する。
