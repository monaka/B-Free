  1. Firstly

    We start B-Free project. This document is our decisive announcement. and, This document declares an aim of a project.

  2. Purpose of a project

    A purpose of B-Free project is to make OS that conforms to BTRON specification. We call it B-Free OS, and We do design and coding of a specification. A base of B-Free OS is a micro-kernel. It is a complete multi-task OS. It is OS for CPU more than 32 bit use.

    We paid attention to the next three facts, and We got cheerful. We did the result, decision that makes BTRON. Also, we do for a ground, because a project is successful.

    • The a few number of people even can make ITRON. It is reported how many.
    • BTRON that uses ITRON for a kernel is reported.
    • Linux is there, now. :-)

    B-Free OS that we make is freely distributed, in future. Of course, a source code also is included in it. B-Free OS is used in the world, and for that an understanding person and cooperation person to TRON project increase, we wish it.
