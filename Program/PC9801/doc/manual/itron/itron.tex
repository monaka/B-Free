\section{はじめに (本ドキュメントの内容)}

本ドキュメントは、{\tt B-Free} \/ OS の中心核となる ITRON についての情
報を記述したものです。

B$-$Free OS を作成するときの ITRON(中心核) のコンパイルする方法および中心核
の上に載るプログラム(周辺核等)をどのように中心核と一緒に動かすのかという方法
が書いてあります。

本ドキュメントの構成は以下のとおりです。

\begin{itemize}
\item ITRON の make 方法
\item ITRON 上に載るアプリケーションの作成方法
\item ITRON で使用できるシステムコールおよびライブラリ関数
\end{itemize}
