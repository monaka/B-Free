\function{vopn\_dev  --- デバイスの登録}


\begin{lablist}

\labitem{呼び出し形式}

{\tt ER vopn\_dev (B *name, IO\_TYPE type, ID *rid)}

\labitem{引数}

\begin{lablist}
  \labitem {name} {デバイス名。}
  \labitem {type} {I/O の種類。}
  \labitem {rid}  {オープンしたデバイスと通信するためのメッセージポー
    ト ID}
\end{lablist}

\labitem{返り値}

以下のエラー番号が返る。

\begin{lablist}
\labitem {E\_OK}     システムコールは正常に終了した
\labitem {E\_NOMEM}  システム管理領域が足りない(デバイスドライバの管理
テーブルがオーバーフローした)。
\end{lablist}


\labitem{機能}

デバイスをオープンし、そのデバイスに対する読み書き用のメッセージポート
 ID を返します。


\end{lablist}
