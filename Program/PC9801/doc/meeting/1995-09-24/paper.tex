\subsection*{今後の予定}

\begin{enumerate}
\item 最小限のデバイスドライバの作成

  \begin{enumerate}
  \item コンソールドライバ(キャラクタ表示) (完成)
  \item キーボードドライバ                 (完成)
  \item FD ドライバ \hfill \\
    Boot で使っているものに HD/HC のモード切換え機能を追加したもの。
  \item HD ドライバ (SCSI) \hfill \\
    FreeBSD(98) を参考に作成。Boot にも移植。
  \end{enumerate}

\item CLI (Command Line Interface) プログラムの作成

現在 ITRON 中に埋め込まれている CLI を独立したプログラムに分離する。
諸々のサーバのテストに使用する。

また、1 と 2 の作業と並行して以下の作業も行う。

\item BTRON マネージャ (周辺核) の I/F 決め

周辺核の送受信するメッセージフォーマットを決める。
デバイス・マネージャについては、メッセージはすでに決まっているので、作っ
てしまった方が良い？
\end{enumerate}
