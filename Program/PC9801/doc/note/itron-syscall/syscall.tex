\section{はじめに}


ITRON は、このドキュメントに示すシステムコールを提供する。
これらのシステムコールは、マイクロ ITRON V3.0 を元に最小限必要となるも
のを選択したものである。よって ITRON のフルセットの機能全てはそなえて
いない。

以下では、ITRON (中心核) のシステムコールを 表 \ref{tab:list} の分類にし
たがって、説明する。

\function { cre\_tsk --- タスク生成 }

\begin{lablist}
  \labitem {呼び出し形式} 

  \#include <itron.h>

  cre\_tsk (ID tskid, T\_CTSK *tskpkt);

  \labitem {引数}

  \begin{lablist}
    \labitem {tskid}  生成するタスクの ID
    \labitem {tskpkt} タスクの生成情報。以下の構造体にて指定する。

    \begin{verbatim}
typedef struct t_ctsk
{
  VP            exinf;          /* 拡張情報             */
  ATR           tskatr;         /* タスク属性           */
  FP            startaddr;      /* タスク起動アドレス   */
  PRI           itskpri;        /* タスク起動時優先度   */
  INT           stksz;          /* スタックサイズ       */
  ADDR_MAP      *addrmap;       /* アドレスマップ       */
                                /* 仮想メモリのマップ   */
                                /* テーブル             */
} T_CTSK;
    \end{verbatim}
  \end{lablist}

  \labitem {機能}  新しいタスクを生成する。

  \labitem {返り値}  以下の返り値が返る。

  \begin{lablist}
    \labitem {E\_OK} システムコールは正常に終了した。
  \end{lablist}
\end{lablist}
